## Usage

### Install Dependencies
```bash
$ mvn clean install && npm install
```

### Build War & Deploy
You can deploy you war file in `target/agile-pratice.war` to your webapp container.(Tomcat, JBoss, Weblogic...)

### Frontend UnitTest
```bash
$ npm test
```
